# Weblinks Manager (B2)

## Description
A simple full-stack web application to manage weblinks.

Users can:
- Add weblinks
- Edit weblinks
- Delete weblinks
- View all weblinks

## Technologies Used
- Node.js
- Express
- HTML, CSS, JavaScript
- Jest & Supertest (Testing)
- APIDoc (API Documentation)

 
## How to Run

1. Install dependencies:
```
npm install
```

2. Start the server:
```
npm start
```

3. Open in browser:
```
http://localhost:3000
```

## Running Tests
```
npm test
```

## API Documentation
Open this file in browser:
```
apidoc/index.html
```
